#TrilhaChallenge

O #TrilhaChallenge é uma experiência prática onde você vai transformar um layout profissional em site funcional — do jeitinho que acontece no mercado.

Você vai receber um layout exclusivo criado no Figma, junto com orientações e prazo para colocar a mão no código. Seu objetivo? Reproduzir o layout com fidelidade, aplicando boas práticas de HTML, CSS e JavaScript (ou a stack que você estiver aprendendo).

E mais: esse exercício é um passo real rumo ao seu portfólio profissional.
